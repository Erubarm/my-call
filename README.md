# CALL — WebRTC звонилка

Простое веб-приложение для видео/аудио звонков. Без регистрации, без баз данных — переходишь по ссылке и сразу в комнате.

## Структура

```
call/
├── public/
│   ├── index.html   — интерфейс
│   ├── style.css    — стили
│   └── app.js       — WebRTC логика (PeerJS)
├── server.js        — Express + PeerJS сигнальный сервер
├── package.json
└── .gitignore
```

## Режим 1 — без своего сервера (GitHub Pages)

Просто открой `public/index.html` или задеплой папку `public/` на GitHub Pages.  
Сигнализация идёт через публичный `0.peerjs.com`.

## Режим 2 — свой сервер

### Установка
```bash
npm install
npm start
```

Сервер запустится на `http://localhost:9000`.

### Переключить фронт на свой сервер

В `public/app.js` раскомментируй блок `Свой PeerJS сервер` и замени `host`.

### Для внешнего доступа без белого IP — используй ngrok

```bash
# Установка (Windows)
winget install ngrok

# Запуск туннеля
ngrok http 9000
```

ngrok выдаст публичный URL вида `https://xxxx.ngrok-free.app` — это и есть адрес твоего сервера.

## Функции

- 🎤 Голосовой звонок (обязательно при входе)
- 📷 Камера (опционально, включается по кнопке)
- 🖥️ Демонстрация экрана (опционально)
- 👥 Несколько участников (mesh P2P)
- 📊 Визуализация звука
- 🔗 Ссылка-комната (хэш в URL)
